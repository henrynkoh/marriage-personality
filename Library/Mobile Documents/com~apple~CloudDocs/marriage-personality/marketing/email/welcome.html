앱을 시작하는 방법

결혼인성 앱은 웹 기반 앱으로, 별도의 설치 없이 브라우저에서 바로 사용할 수 있습니다:

  1. 아래 버튼을 클릭하여 앱에 접속하세요.
  2. "설문 시작하기" 버튼을 클릭하세요.
  3. 5개의 간단한 질문에 솔직하게 답변하세요.
  4. 결과를 확인하고, 관계 개선 팁을 살펴보세요.

주요 기능

5분 만에 완료되는 간편한 설문
가족 중심, 갈등 해결, 재정 관리, 의사소통, 독립성 등 5가지 핵심 영역 평가

오프라인에서도 사용 가능한 PWA(Progressive Web App)

PWA로 설치하기

결혼인성 앱은 PWA 기술을 활용해 모바일 홈 화면에 설치할 수 있습니다:

iOS: Safari에서 공유 버튼을 탭한 후 "홈 화면에 추가"를 선택하세요.

Android: Chrome에서 메뉴 버튼을 탭한 후 "홈 화면에 추가"를 선택하세요.

궁금한 점이 있으시면 언제든지 contact@gyeolhoninseong.kr로 문의해주세요.
